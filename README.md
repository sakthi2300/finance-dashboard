# 💰 Finance Dashboard UI

A modern and interactive finance dashboard built using Angular to help users track financial activity, analyze spending, and manage transactions efficiently.

---

## 🚀 Features

### 📊 Dashboard
- Total Balance, Income, Expenses summary
- Income vs Expenses trend chart
- Category-wise expense distribution

### 📋 Transactions
- View all transactions in a structured table
- Search, filter, and sort transactions
- Add, edit, and delete transactions (Admin only)
- Export data (PDF)

### 🔍 Insights
- Highest spending category
- Monthly income vs expenses comparison
- Savings rate analysis

### 🔐 Role-Based UI
- Admin:
  - Full access (add/edit/delete/export)
- Viewer:
  - Read-only access

### 🌙 Dark Mode
- Toggle between light and dark themes

---

## 🛠️ Tech Stack

- Angular 17 (NgModule)
- RxJS (state management)
- Highcharts (data visualization)
- Handsontable (data grid)
- Tailwind CSS / Bootstrap
